React Samples for MindFusion.Scheduling for JavaScript, version 1.0


1. Create React App

	npm install -g create-react-app

	create-react-app my-app

	cd my-app


2. Install MindFusion.Scheduling for JavaScript in the 'my-app' folder

	npm install scheduler-library


3. Copy the contents of Samples\React folder into my-app folder


4. Run the sample application

	npm start
